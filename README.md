# Timoty Drinks & Kitchen — Sito verticale

Sito ristorante **premium** per **Timoty** (Roma, Via Torino 46). Riferimento strutturale e qualitativo: **Rosemary** (`verticals/rosemary`).

## Obiettivo
- Sito moderno, ordinato, forte commercialmente.
- Hero forte, sezioni pulite, prenotazione evidente, gallery atmosfera, mobile-first.
- Identità Timoty: cucina genuina, healthy, veg/vegetariano/gluten free, km 0, Roma centro.

- **Implementazione:** [docs/TIMOTY_IMPLEMENTATION_PLAN.md](docs/TIMOTY_IMPLEMENTATION_PLAN.md) — piano step-by-step, checklist file.

- Next.js, TypeScript.
- Stessa qualità e pattern di Rosemary; design system e copy dedicati a Timoty.
- Opzionale: Supabase per prenotazioni/CRM come Rosemary.
